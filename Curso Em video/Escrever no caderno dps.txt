CSS
Estilo em CSS
h1 {

   font-family:arial;  
   font-size: 20pt;
   color:blue;
    }
h1 -> é o seletor.
{ } -> abertura e fechamento.
font-family:arial; -> a linha inteira é adeclaração    
color: -> propriedade 
blue;  -> valor    
Toda declarção tem que ter ; no final
CSS tem seletores dentro dos seletores tem declarações 
e cada declaração é um par de propriedade e valor
